ГИТ_ШПОРА

Порядок команд:
git init (инициализация)
git add . (добавление всех файлов)
git commit -m 'Первый коммит' (сохранение изменений)
git remote add origin <ВАША ССЫЛКА на ВАШ удалённый гитхаб> (связывание с гитхабом)
git branch -M main (переименование ветки, если ветка называется main, то не надо)
git push -u origin main (отправление изменений)

Дополнительные команды:
git status - просмотр информации о гит-репозиторие
cd <ПУТЬ> - перемещение в папку по заданному пути
